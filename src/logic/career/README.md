# logic/career

キャリア全体の評価指標（phase 判定とは別の、総合スコア系）。

- `clearScore.ts` キャリアの「到達度」スコア算出
- `analysis.ts` 観測スタンス、保存推奨、資料館フィルタ、二人比較で共有する派生分析

## `analysis.ts` の責務

- UI文言だけを作らず、数値指標・表示ラベル・判定理由・フィルタ用キーを分けて返す
- 結果画面、資料館、比較機能が同じ計算を使う
- 将来の類似検索、世代比較、宿敵分析を追加するときの入力モデルになる
- 番付推移比較、勝率推移比較、類似キャリア検索、保存済み世代比較もここから派生する
- DBには重い派生値を固定保存せず、保存済み `finalStatus` から再計算する
