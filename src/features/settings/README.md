# settings

設定・データ管理画面。アプリケーション設定と、IndexedDB のデータ管理（削除等の
破壊的操作を含む）を担当します。

## 責務

- 表示・挙動に関する設定 UI
- 保存データのエクスポート/削除などの管理操作

## 主要ファイル

- `components/SettingsScreen.tsx` 画面本体

## 依存

- `src/logic/persistence/` DB 操作

## 注意

- 破壊的操作を含むため、確認モーダルで誤操作を防いでください。
- DB 名の変更（例: `sumo-maker-v13`）が入ると旧データが読めなくなります。
